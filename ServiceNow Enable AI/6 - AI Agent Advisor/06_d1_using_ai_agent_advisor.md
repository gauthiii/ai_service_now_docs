### Result

AI Agent Advisor runs the analysis according to the configured filters and schedule. After the
analysis completes, automation opportunities appear on the Now Assist Center home page and
on the Automation opportunities page.

### What to do next

View your automation opportunities on the home page. For more information, see View your
automation opportunities.

### Using AI Agent Advisor

Use AI Agent Advisor to automatically discover automation opportunities in your instance and
deploy AI agents to implement the automations.
