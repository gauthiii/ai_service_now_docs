Create a labeling project template

You can create a template for labeling projects to quickly assign labeling tasks in a labeling project.

Before you begin Role required: sn_data_kit.admin

Procedure

1.Navigate to All > Now Assist Data Kit > Home.

Create a project or select a project that has already been published. 2.

Select template. 3. Create

4.Give the template a name and select Create. Your template will now appear in the list of available templates.

Add ground truth to a labeling project

Ground truth is the real-world data that is used to train and test AI models.

Before you begin Role required: sn_data_kit.admin

Procedure

1.Navigate to > > Home. All Now Assist Data Kit

2. Create a project using the AI generated ground truth template or the manual ground truth template. For more information about creating a labeling project from a template, see Create labeling project from template.

Note: If you select AI-generated ground truth, tasks are created with pre-filled labels based on setup instructions.
