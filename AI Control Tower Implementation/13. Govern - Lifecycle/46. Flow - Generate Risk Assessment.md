## What Is This?

"Flow: Generate Risk Assessment" is one of the governance lifecycle flows in the AI Control Tower. On this page the subtopic appears as a heading only, introducing the flow that generates a risk assessment as part of the governance process.

## How to Use This in Implementation

This page only establishes the heading for the Generate Risk Assessment flow within the Govern lifecycle. Implementers should treat this as the entry point to the risk-assessment generation flow; the actual step-by-step configuration of this flow continues beyond page 186 and is not detailed here. When wiring up the governance lifecycle, sequence this flow after the analyst-assignment flow, which itself triggers once the AI system impact assessment is closed.
