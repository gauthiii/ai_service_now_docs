## What Is This?

This subtopic provides an illustrative worked example of the inputs used to compute AI-assisted productivity value within AI Control Tower. It shows sample metric values (Usage, Acceptance Rate, Time Saved) and the resulting outcome interpretation. It fits into the Measure - Value workstream as a concrete reference for how productivity value is derived from the underlying metrics.

---

## Sample Calculation

The example uses the productivity formula:

`Productivity = Usage × Time Saved per Invocation × Acceptance Rate`

| Metric | Value |
| --- | --- |
| Usage | 2,250 |
| Acceptance Rate | 30% |
| Time Saved | 5 minutes |

**Outcome:** Represents aggregate productivity gained from AI-assisted interactions.

---

## How to Use This in Implementation

Implementers use this illustrative example as a template for assembling the inputs needed to calculate AI productivity value: capture Usage (number of AI capability invocations), Acceptance Rate (percentage of AI outputs accepted), and Time Saved per invocation. Apply the formula `Productivity = Usage × Time Saved per Invocation × Acceptance Rate` to derive aggregate productivity gained from AI-assisted interactions. The values shown here are illustrative only and should be replaced with actual measured data from the environment.
