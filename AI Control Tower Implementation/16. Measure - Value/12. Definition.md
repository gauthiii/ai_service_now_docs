## What Is This?

This subtopic defines what Value Templates are within AI Control Tower's value measurement capability. Value Templates establish the calculation logic that AICT uses to measure AI-driven productivity and value realization across AI assets.

---

## Value Templates Definition

Value Templates define the calculation logic used to measure AI-driven productivity and value realization.

---

## How to Use This in Implementation

When implementing value measurement in AICT, Value Templates serve as the foundation for how productivity and value realization are calculated for AI assets. An implementer selects or configures a Value Template to apply consistent calculation logic when measuring AI-driven value. This definition is the entry point; subsequent steps involve choosing out-of-box templates (such as Third-party Assets, ServiceNow Skills, ServiceNow Agents, and Agentic Workflows) or creating new templates via Workspaces > AI Control Tower > AI Asset > Productivity > Value templates.
