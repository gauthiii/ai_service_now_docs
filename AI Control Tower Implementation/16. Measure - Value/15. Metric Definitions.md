## What Is This?

This subtopic defines the core metrics used by AI Control Tower to measure the value and productivity of AI assets. It explains each metric, how the metrics behave (where their values come from and how they are configured), and provides an illustrative sample calculation. These metrics feed the overall productivity computation: Productivity = Usage × Time Saved per Invocation × Acceptance Rate.

---

## Metric Definitions

| Metric | Definition |
|---|---|
| Persona | User role associated with the AI asset (e.g., Agent, Developer, Requestor) |
| Usage | Number of times an AI capability is invoked (tracked daily via PA indicators) |
| Time Saved | Estimated reduction in manual effort per invocation |
| Acceptance Rate | Percentage of AI outputs accepted by users |

---

## Metric Behavior

- **Usage**
  - Derived from daily PA indicators
  - Requires completion before scheduled value calculations
- **Time Saved**
  - Can be:
    - Static value (e.g., 15 minutes)
    - Indicator-based
- **Acceptance Rate**
  - Formula: Accepted Outputs ÷ Total Outputs
  - Configurable as static or indicator-driven

---

## Sample Calculation (Illustrative)

| Metric | Value |
|---|---|
| Usage | 2,250 |
| Acceptance Rate | 30% |
| Time Saved | 5 minutes |

**Outcome:** Represents aggregate productivity gained from AI-assisted interactions.

---

## How to Use This in Implementation

When implementing value measurement, ensure the Usage metric is sourced from daily Performance Analytics (PA) indicators and that those indicators complete before scheduled value calculations run. Configure Time Saved either as a static value (such as 15 minutes per invocation) or as an indicator-based value depending on available data. Set Acceptance Rate using the formula Accepted Outputs ÷ Total Outputs, again as either a static or indicator-driven value. Associate each AI asset with the appropriate Persona. These metrics combine into the productivity formula Productivity = Usage × Time Saved per Invocation × Acceptance Rate to represent aggregate productivity gained from AI-assisted interactions.
