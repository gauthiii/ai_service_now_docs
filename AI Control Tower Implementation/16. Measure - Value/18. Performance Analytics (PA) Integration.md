## What Is This?

Performance Analytics (PA) Integration describes how AI Control Tower's value and productivity metrics are sourced from and depend on ServiceNow Performance Analytics. Usage and other value metrics are tracked through daily PA indicators, which must complete before scheduled value calculations run. This subtopic introduces that integration within the Measure - Value area of the implementation.

## How to Use This in Implementation

Implementers should ensure Performance Analytics is configured and that the relevant daily PA indicators (notably the Usage indicator) are running and completing on schedule, since scheduled value calculations depend on these indicators having finished. PA Integration is therefore a prerequisite for accurate productivity and value reporting in AI Control Tower.
