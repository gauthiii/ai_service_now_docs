## What Is This?

This subtopic defines the purpose of the Multi-Instance Framework (MIF) Configuration within the value measurement capabilities of AI Control Tower. It explains why MIF is configured and the outcome it enables for value reporting across environments.

---

## Purpose of Multi-Instance Framework (MIF) Configuration

Enables value aggregation across production and sub-production environments.

---

## How to Use This in Implementation

When measuring value in AICT across more than one instance, configure the Multi-Instance Framework so that value data from sub-production environments can be aggregated together with production. This is the prerequisite goal that the MIF setup steps (registering the production instance, configuring instance relationships, and adding sub-production instances) are meant to achieve.
