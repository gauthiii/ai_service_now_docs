## What Is This?

System Behavior and Automation describes the scheduled jobs and dependencies that power the Value measurement capability in AI Control Tower. It explains how usage and productivity metrics are calculated, how baseline data is generated for new instances, and what the value dashboards depend on. This fits into the Measure - Value implementation phase, where AICT computes and surfaces the business value of AI usage.

---

## System Behavior and Automation

- **Daily Jobs**
  - Calculate usage and productivity metrics
- **Historical Data Job**
  - Generates 30 days of baseline data for new instances
- **Indicator Dependency**
  - Value dashboards depend on successful PA indicator execution

---

## How to Use This in Implementation

Ensure the daily jobs and the historical data job run successfully, as they produce the usage/productivity metrics and the 30-day baseline data that the value dashboards rely on. Because value dashboards depend on successful Performance Analytics (PA) indicator execution, verify PA indicators run without error; if PA indicator execution fails, the value dashboards will not populate correctly.
