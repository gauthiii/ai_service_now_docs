## What Is This?

Release Compatibility for the Value Management capability documents which AI Control Tower (AICT) release introduced the Value features and the corresponding version availability. It helps implementers confirm that their environment is on a release that supports Value Management before configuring Value Templates and dashboards.

---

## Version Availability by Release

| Release | Version Availability |
| --- | --- |
| May | Innovation Lab only (YP3) |
| July | YP6, ZP1 |
| December | YP11, ZP4 |

---

## How to Use This in Implementation

Use this table to verify your instance is on a supported release before enabling Value Management. The May release exposes the capability in the Innovation Lab only (YP3), while general availability follows in the July (YP6, ZP1) and December (YP11, ZP4) releases. Confirm the platform version matches one of these before configuring Value Templates and dashboards.
