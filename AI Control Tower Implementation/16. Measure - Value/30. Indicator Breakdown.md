## What Is This?

Indicator Breakdown is an operational consideration within the AI Control Tower Value measurement capability. It addresses the need to break indicators down at the individual asset level so that value reporting remains accurate. It fits into implementation as part of the operational guidance for ensuring trustworthy value dashboards and metrics.

---

## Indicator Breakdown

This subtopic falls under the "Operational Considerations" heading and captures the requirements for breaking indicators down by asset:

- Required for asset-level accuracy
- Prevents aggregation errors across assets

---

## How to Use This in Implementation

When configuring value measurement, ensure indicators are broken down at the asset level. This is required to achieve accurate asset-level reporting and to avoid aggregation errors that can occur when values are combined across multiple assets.
