---

## What Is This?

Value Templates define the calculation logic used to measure AI-driven productivity and value realization within AI Control Tower. They provide the standardized formulas applied to AI assets so that value can be measured consistently. AICT ships with out-of-box templates and also allows implementers to add new custom templates.

---

## Definition

Value Templates define the calculation logic used to measure AI-driven productivity and value realization.

## Out-of-Box Templates

- Third-party Assets
- ServiceNow Skills
- ServiceNow Agents
- Agentic Workflows

## Adding a New Template

1. Go to: Workspaces > AI Control Tower -> AI Asset -> Productivity -> Value templates
2. Click: New Template

---

## How to Use This in Implementation

Use the out-of-box Value Templates (Third-party Assets, ServiceNow Skills, ServiceNow Agents, Agentic Workflows) to apply standardized value-realization calculation logic to the corresponding AI asset types. When the out-of-box templates do not cover a needed measurement, create a custom template by navigating to Workspaces > AI Control Tower -> AI Asset -> Productivity -> Value templates and clicking New Template.
