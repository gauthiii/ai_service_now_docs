---

## What Is This?

This subtopic identifies the specific jobs and scripts that must be running for AI Value data to become visible in dashboards. It breaks out the dependency chain clearly so implementers know which background processes are prerequisites for populating value data. It fits within the AI Value measurement workflow, alongside data initialization and post-clone data availability troubleshooting.

---

## Required Jobs

Break this out clearly:

- PA indicator jobs must run
- `aivalue.valuedashboard.HistoricalJob`
- AIValue historical script must be active

---

## How to Use This in Implementation

To ensure AI Value data is visible, confirm that the Performance Analytics (PA) indicator jobs are executing, that the `aivalue.valuedashboard.HistoricalJob` job is present and running, and that the AIValue historical script is active. These three items form the dependency chain for data visibility; if any is missing or inactive, value dashboards may not populate.
