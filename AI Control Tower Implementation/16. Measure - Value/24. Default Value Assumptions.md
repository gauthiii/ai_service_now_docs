---

## What Is This?

Default Value Assumptions are the out-of-the-box (OOB) baseline figures AI Control Tower uses to establish the baseline logic for value measurement. They define the starting assumptions for calculating productivity and value metrics before any customization. They should be established first so the value calculations have a defined baseline, and they can be overridden later via Value Templates.

---

## Default Value Assumptions

Place this first to establish baseline logic:

- 15-minute time saved
- 50% acceptance rate
- Clarify these are:
  - OOB template assumptions
  - Overridable via Value Templates

---

## How to Use This in Implementation

When configuring value measurement in AI Control Tower, establish the Default Value Assumptions first so the baseline logic is in place before other configuration. The OOB assumptions (15 minutes saved per interaction and a 50% acceptance rate) come from the out-of-the-box template and can be adjusted to fit your organization by overriding them through Value Templates.
