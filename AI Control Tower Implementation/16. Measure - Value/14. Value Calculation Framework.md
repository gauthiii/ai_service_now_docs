## What Is This?

The Value Calculation Framework is the mechanism AI Control Tower (AICT) uses to quantify the productivity value delivered by AI systems. It applies a standardized formula so that productivity is measured consistently across AI skills, AI agents, and AI workflows. This sits within the Measure - Value area of AICT, supporting value templates and default template mappings.

## How to Use This in Implementation

When implementing value measurement in AICT, understand that productivity is computed through a single standardized formula applied across AI systems. This framework works together with the value templates and default template mapping configuration (Configuration → Controls → Value templates → Assign Default templates), where templates are mapped by template name and vendor, and—when the vendor is ServiceNow—by AI system category (AI skill, AI agent, or AI workflow). The specific formula details should be reviewed on the continuation of this section.
