## What Is This?

This subtopic lists the items that must be completed before configuring a Trace Collector in AI Control Tower (AICT). Trace Collectors establish connections with cloud hyperscalers to collect structured trace data from externally hosted AI agents and route it to ServiceNow's observability service. Completing these prerequisites is essential, since incomplete prerequisites are the most common cause of failed connections.

---

## Prerequisites

Complete the following before configuring a trace collector. Incomplete prerequisites are the most common cause of failed connections.

| Requirement | Details | Owner |
|---|---|---|
| Cloud Credentials — Authentication credentials for the target cloud provider (AWS, GCP, or Azure) | Must be pre-created in the ServiceNow Credentials module before starting configuration. | Platform Admin |
| Active MID Server — ServiceNow MID Server installed, validated, and in Active status | The MID Server acts as a secure intermediary between ServiceNow and the cloud provider. See MID Server Installation docs. | Platform Admin |
| Collection Frequency — Defined polling interval in minutes | Work with AI agent owners to align on an appropriate collection frequency before configuring. | AI Program Lead |
| Credential Alias (Feb Release only) — API Key credential alias for the observability service | Required only for the February Innovation Lab release. Not required for the March release onwards. | Platform Admin |
| AWS Region (AWS only) — Specific AWS region where AI agents are deployed | Required only for AWS AgentCore connections. Confirm the region with your cloud team before proceeding. | Cloud/Platform Team |

---

## How to Use This in Implementation

Before setting up a Trace Collector, the Platform Admin should pre-create the target cloud provider's authentication credentials in the ServiceNow Credentials module and ensure an MID Server is installed, validated, and in Active status to serve as the secure intermediary. The AI Program Lead works with AI agent owners to agree on a polling/collection frequency in minutes. Release-specific items apply conditionally: the API Key Credential Alias is needed only for the February Innovation Lab release (not March onwards), and the AWS Region is needed only for AWS AgentCore connections. Because incomplete prerequisites are the most common cause of failed connections, confirm all applicable items are satisfied before starting configuration.
