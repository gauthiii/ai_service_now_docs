## What Is This?

This subtopic lists the external and platform documentation references that support the Observe - Trace Collectors implementation. It points implementers to the source documentation for MID Server installation, the AI Control Tower workspace, credentials configuration, and AI Evaluations. These links serve as the authoritative follow-on resources when configuring trace collection and related components.

## How to Use This in Implementation

Use these references as navigation pointers while implementing trace collectors. The MID Server link provides the installation procedure required for trace collection that depends on a MID Server. The Credentials Module path (ServiceNow Platform > Connections & Credentials > Credentials) is where credentials are managed, and the AI Evaluations path (AI Control Tower > Configurations > AI Evaluations) is where related evaluation configuration is found. The AI Control Tower Overview points to the workspace within your own ServiceNow instance.
