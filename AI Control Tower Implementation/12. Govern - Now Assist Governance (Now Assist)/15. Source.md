## What Is This?

"Source" identifies the originating data object used during the **Sync Prompt Assets** step of Now Assist governance synchronization. It specifies which ServiceNow table the prompt asset data is read from before being synced into the AI Control Tower asset model. This fits into the broader prompt-asset synchronization process that creates and maps prompt digital assets and product models.

---

## Source (Sync Prompt Assets)

The source for syncing prompt assets is:

- *Generative AI Configurations* `$sys_generative_ai_config$` (Prompt Templates)

This is the originating record set (Prompt Templates) that the prompt asset synchronization process iterates through.

---

## How to Use This in Implementation

When configuring the Sync Prompt Assets step, point the synchronization at the **Generative AI Configurations** table (`$sys_generative_ai_config$`), which holds the Prompt Templates. The process then iterates through these prompt configurations and, for each prompt, identifies or creates the associated AI Prompt Digital Asset and AI Prompt Product Model. Ensure the `sys_generative_ai_config` records exist and are populated before running the sync.
