## What Is This?

Product Model Attributes define the data fields captured for an AI model's product model record within Now Assist governance. They are organized into two groups — Model Configuration and Model Information — that together describe how a model is configured and the descriptive metadata associated with it. These attributes are populated when configuration and model information are mapped to the digital asset.

---

## Model Configuration

The Model Configuration group includes the following attributes:

- Name
- Model Category
- Supported Languages
- Context Window

---

## Model Information

The Model Information group includes the following attributes:

- Documentation
- Manufacturer
- Parameters
- Model Size (MB)
- Source
- Deployment Guideline
- Training Procedure

---

## How to Use This in Implementation

When mapping configuration and model information to a model digital asset, populate these attributes on the AI Product Model record. Use the Model Configuration attributes (Name, Model Category, Supported Languages, Context Window) to capture how the model operates, and the Model Information attributes (Documentation, Manufacturer, Parameters, Model Size, Source, Deployment Guideline, Training Procedure) to record descriptive metadata. These attributes support the Model Digital Asset Updates (Model Card, Required Infrastructure, Evaluation Metrics Report) that accompany the governance process.
