# 6. Sync AI Use Cases

---

## What Is This?

Sync AI Use Cases is one of the synchronization processes within Now Assist Governance. It maps AI use case metadata into the associated AI System by syncing with the AI System Component Product Model. This ensures that AI use cases governed in the AI Control Tower are reflected as part of the AI system's representation.

---

## Process

The synchronization follows these steps:

- Identify associated AI System
- Sync with AI System Component Product Model
- Map use case metadata to the AI system

---

## Mapped Attributes

The following attributes are mapped during synchronization:

- Name
- Description
- Model Category
- Documentation
- Manufacturer

### Note

Prompt product models are also synchronized for associated AI Prompt Digital Assets.

---

## How to Use This in Implementation

When configuring AI use case governance, ensure each use case has an associated AI System so the sync can identify it and map metadata (Name, Description, Model Category, Documentation, Manufacturer) into the AI System Component Product Model. Be aware that the synchronization additionally propagates prompt product models for any associated AI Prompt Digital Assets, so those assets should be linked correctly to obtain complete synchronization results.
