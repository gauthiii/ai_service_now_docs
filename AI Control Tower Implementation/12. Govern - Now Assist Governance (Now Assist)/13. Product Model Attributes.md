## What Is This?

Product Model Attributes define the fields populated on the AI Product Model record created during asset synchronization. In this context, they specify the attributes mapped when syncing model assets into the AI Control Tower governance data model. These attributes establish the descriptive metadata for each synced product model.

---

## Product Model Attributes

The following attributes are mapped for the product model:

- Name
- Description
- Model Category

---

## How to Use This in Implementation

When configuring the synchronization that creates or updates AI Product Model records, populate these three attributes (Name, Description, Model Category) so each synced model carries consistent descriptive metadata. These fields feed the governance views in the AI Control Tower. Ensure source data provides values for each attribute before running the sync.
