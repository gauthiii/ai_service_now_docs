## What Is This?

"Default Recommendation" is part of the Configuration Guidance for the AICT asset synchronization scheduled job. It states the recommended posture for the scheduled job — keeping it enabled — and the benefits this provides. It fits into implementation as the baseline guidance for whether automatic discovery/synchronization should remain active.

---

## Default Recommendation

It is strongly recommended to keep the scheduled job enabled to ensure:

- Continuous inventory accuracy
- Alignment between deployed AI capabilities and governance records
- Reduced operational overhead

---

## How to Use This in Implementation

When configuring the AICT asset synchronization scheduled job, leave it enabled by default. Doing so keeps the asset inventory continuously accurate, keeps deployed AI capabilities aligned with governance records, and minimizes operational overhead compared to manual maintenance. Only deviate from this default if there is a specific reason to disable automatic discovery (covered separately under Optional Deactivation).
