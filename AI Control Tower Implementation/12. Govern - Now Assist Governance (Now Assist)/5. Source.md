## What Is This?

"Source" defines the origin configuration table that the "Sync Now Assist AI Assets" job reads from when performing the **Sync Model Assets** step of automated AI asset synchronization. It identifies where AICT looks to discover Now Assist generative AI model configurations so they can be mapped into the AICT inventory. This sits within AICT's automated AI asset synchronization for Now Assist.

---

## Source for Sync Model Assets

The source for the **1. Sync Model Assets** step of the Now Assist synchronization process is:

- *Generative AI Model Configurations* — `$sys_generative_ai_model_config$`

This is the configuration table that the synchronization job iterates through to identify AI model assets and their associated metadata for the Sync Model Assets step.

---

## How to Use This in Implementation

When configuring or troubleshooting automated AI asset synchronization for Now Assist, implementers should know that the Sync Model Assets step draws its model data from the `sys_generative_ai_model_config` table (Generative AI Model Configurations). Ensure this table is populated with the live Now Assist generative AI model configurations, since the scheduled "Sync Now Assist AI Assets" job reads from this source to create or update the corresponding AICT digital assets and product models. If model assets are missing from the AICT inventory, verify that the source configurations exist in `sys_generative_ai_model_config`.
