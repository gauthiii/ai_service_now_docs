## What Is This?

Sync AI Tools is part of the Sync Agentic AI Assets process, which synchronizes Agentic AI assets — AI Tools, AI Agents, and AI Use Cases — derived from Now Assist configurations. This step specifically maps AI Tool records from their source table into the enterprise AI tool target table within AI Control Tower governance.

---

## Sync AI Tools

Agentic AI synchronization includes AI Tools, AI Agents, and AI Use Cases derived from Now Assist configurations.

**Source:** `$sn_aia_tool$`
**Target:** `$sn_ent_ai_tool$`

### Mapped Attributes

- Name
- Description
- Type

---

## How to Use This in Implementation

When synchronizing Agentic AI assets from Now Assist, the AI Tool records are read from the source table `sn_aia_tool` and synced into the enterprise AI tool target table `sn_ent_ai_tool`. During this sync, the Name, Description, and Type attributes are mapped from source to target so that AI Tools are represented and governed within AI Control Tower. This sync is performed alongside the related AI Agents and AI Use Cases syncs that make up the broader Agentic AI asset synchronization.
