# Required Roles

> ServiceNow AI Control Tower (AICT) — Implementation Guide
> Section: Discover - Discovery → Required Roles

---

## What Is This?

This subtopic specifies the roles required to configure and use the AI Service Graph Connector for Amazon, which discovers AI assets and usage data from AWS services into ServiceNow AI Control Tower. These roles grant the permissions needed to set up the Amazon AI connection and run discovery within the Discover practice area.

---

## Required Roles

The following roles are required for the AI Service Graph Connector for Amazon:

- `sn_ai_disc.discovery_admin`
- `sn_cmdb_int_util.sgc_admin`

---

## How to Use This in Implementation

Before configuring the AI Service Graph Connector for Amazon, ensure the implementing user is granted both the `sn_ai_disc.discovery_admin` and `sn_cmdb_int_util.sgc_admin` roles. These roles are prerequisites for creating the Amazon AI connection and performing discovery of AI assets (AI systems, agents, models, tools, prompts, and usage records) from AWS into ServiceNow AI Control Tower.
