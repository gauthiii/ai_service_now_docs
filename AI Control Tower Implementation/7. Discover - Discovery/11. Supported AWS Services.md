## What Is This?

This subtopic lists the AWS services from which the AI Service Graph Connector for Amazon can discover AI assets and collect usage data into ServiceNow AI Control Tower. It defines the scope of what the Amazon connector can ingest. It fits into the Discover phase, where AI assets and usage data are pulled from external cloud providers into AICT.

---

## Supported AWS Services

The Amazon connector supports discovery and usage data from:

- Amazon Bedrock
- Amazon Bedrock AgentCore
- Amazon SageMaker
- Amazon CloudWatch Logs

---

## How to Use This in Implementation

When configuring the AI Service Graph Connector for Amazon, ensure API access is enabled for the AWS services you intend to connect (Amazon Bedrock, Amazon SageMaker, Amazon CloudWatch, and Amazon Bedrock AgentCore, as applicable) and that the IAM credentials carry the required read permissions for each of these services. Only the services listed above are in scope for discovery and usage data ingestion through the Amazon connector.
