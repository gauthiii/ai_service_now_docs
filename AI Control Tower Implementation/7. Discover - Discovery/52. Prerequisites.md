## What Is This?

This subtopic describes the prerequisites that must be satisfied before creating a LangGraph connection for the AI Service Graph Connector for LangGraph in ServiceNow AI Control Tower. It establishes the API access and configuration needed for AI Control Tower to discover AI assets and usage data from LangGraph and LangSmith. It fits into the discovery setup phase, immediately before a connection is created.

---

## Prerequisites for the LangGraph Connection

Before creating a LangGraph connection, confirm that an API key has been created in LangSmith. The API key must provide access to the workspaces where AI assets and usage data should be discovered.

For self-hosted LangSmith environments, the organization should also confirm the appropriate LangSmith API and host API URLs.

---

## How to Use This in Implementation

To prepare for creating a LangGraph connection, an implementer should:

- Create an API key in LangSmith that grants access to the workspaces containing the AI assets and usage data to be discovered.
- For self-hosted LangSmith environments, identify and confirm the appropriate LangSmith API URL and host API URL.

These prerequisites support the AI Service Graph Connector for LangGraph, which discovers AI assets and usage data (AI systems, models, prompts, tools, agents, and usage records) from LangGraph and LangSmith into ServiceNow AI Control Tower.
