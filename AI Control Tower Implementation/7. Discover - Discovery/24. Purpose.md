## What Is This?

This subtopic states the purpose of the Azure Foundry connection within AI Control Tower discovery. The connection lets AI Control Tower discover Azure Foundry assets and retrieve execution data for supported AI services. It is the entry point for bringing Azure Foundry-managed AI resources into the AICT discovery scope.

---

## Purpose of the Azure Foundry Connection

The Azure Foundry connection enables AI Control Tower to discover Azure Foundry assets and retrieve execution data for supported AI services.

---

## How to Use This in Implementation

Implementers establish the Azure Foundry connection so that AI Control Tower can begin discovering Azure Foundry assets and collecting their execution data. This connection is a prerequisite step before discovered Azure Foundry agents, models, prompts, tools, subcomponents, and execution data become visible in AI Control Tower. Configuration of the connection's prerequisites and the resulting data mapping are covered in the related Azure Foundry subtopics on the same page.
