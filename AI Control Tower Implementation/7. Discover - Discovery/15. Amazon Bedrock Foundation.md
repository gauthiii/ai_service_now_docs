## What Is This?

Amazon Bedrock Foundation is one of the groups of AWS APIs that AI Control Tower uses when discovering AWS AI assets. After the Amazon AI connection is created, AI Control Tower can import AWS AI asset metadata and usage data into the AI inventory and related usage tables. This subtopic lists the specific AICT action names and the underlying AWS APIs they call for foundation model discovery.

---

## AWS APIs Used — Amazon Bedrock Foundation

The following actions map AICT action names to the AWS APIs invoked for the Amazon Bedrock Foundation group.

| Action Name | AWS API |
|---|---|
| `sgawsbedrock_get_inference_profile` | GetInferenceProfile |
| `sgawsbedrock_get_foundation_model` | GetFoundationModel |

---

## How to Use This in Implementation

These actions are used by AI Control Tower after the Amazon AI connection is established. The AWS APIs `GetInferenceProfile` and `GetFoundationModel` are called to retrieve foundation model metadata, which is imported into the AI inventory and related usage tables. Ensure the AWS connection has the permissions required to call these Bedrock APIs so that foundation model assets are discovered correctly. This group sits alongside the Amazon Bedrock Agents and Amazon Bedrock AgentCore Control groups within the broader set of AWS APIs used by discovery.
