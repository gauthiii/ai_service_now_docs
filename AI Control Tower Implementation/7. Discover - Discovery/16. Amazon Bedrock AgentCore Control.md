## What Is This?

Amazon Bedrock AgentCore Control is one of the categories of AWS APIs that AI Control Tower uses when discovering Amazon AI assets. It is listed under the "AWS APIs Used" section, alongside Amazon Bedrock Agents and Amazon Bedrock Foundation. These API groupings define the specific AWS actions that AI Control Tower invokes—after an Amazon AI connection is created—to import AWS AI asset metadata and usage data into the AI inventory and related usage tables.

## Amazon Bedrock AgentCore Control

This subtopic appears as a heading under the "AWS APIs Used" section on page 68. On this page, the heading "Amazon Bedrock AgentCore Control" is shown with no accompanying action-name/AWS-API table or body content (the associated content continues beyond this page).

### Context: AWS APIs Used

After the Amazon AI connection is created, AI Control Tower can import AWS AI asset metadata and usage data into the AI inventory and related usage tables.

The "AWS APIs Used" section organizes the relevant AWS actions into named groups, each presented as a table mapping an **Action Name** to its corresponding **AWS API**. The "Amazon Bedrock AgentCore Control" group is the third such grouping listed on this page, following:

- **Amazon Bedrock Agents** (e.g., `sgawsbedrock_agent` → GetAgent, `sgawsbedrock_list_agent` → ListAgents, and related actions)
- **Amazon Bedrock Foundation** (e.g., `sgawsbedrock_get_inference_profile` → GetInferenceProfile, `sgawsbedrock_get_foundation_model` → GetFoundationModel)

## How to Use This in Implementation

When configuring an Amazon AI connection for AI Control Tower discovery, implementers should be aware that the integration relies on distinct groups of AWS APIs, including the Amazon Bedrock AgentCore Control group. The AWS credentials/role used for the connection must be permitted to invoke the AWS APIs in each enabled group so that AI Control Tower can import the corresponding AWS AI asset metadata and usage data into the AI inventory and related usage tables. The specific Action Name to AWS API mappings for Amazon Bedrock AgentCore Control continue beyond this page.
