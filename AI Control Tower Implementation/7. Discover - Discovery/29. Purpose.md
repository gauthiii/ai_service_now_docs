## What Is This?

This subtopic states the purpose of the Azure Foundry connection within AI Control Tower's Discovery capability. It explains why the connection exists and what it enables AI Control Tower to do with respect to Azure Foundry assets and execution data. It is the introductory framing for configuring the Azure Foundry connection.

---

## Purpose

The Azure Foundry connection enables AI Control Tower to discover Azure Foundry assets and retrieve execution data for supported AI services.

---

## How to Use This in Implementation

An implementer uses this connection to bring Azure Foundry assets and their execution data into AI Control Tower's Discovery scope. Establishing the Azure Foundry connection is the prerequisite step before AI Control Tower can discover the associated Azure Foundry agents, models, prompts, tools, subcomponents, and execution data. Implementers should proceed to confirm the Azure Foundry prerequisites and configure the required import schedules to complete the setup.
