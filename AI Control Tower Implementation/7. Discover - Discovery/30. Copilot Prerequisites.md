## What Is This?

This subtopic introduces the prerequisites for setting up the Microsoft Copilot connection in AI Control Tower. The Copilot connection enables AI Control Tower to discover Copilot Studio agents and retrieve related execution or usage data from the Microsoft environment. It fits within the Discover phase of AICT implementation, alongside other connection types used to ingest external AI assets.

## How to Use This in Implementation

Before configuring the Microsoft Copilot connection, an implementer should review the Copilot prerequisites (continued on the next page) to confirm the required Microsoft environment access and credentials are in place. This connection is what allows AI Control Tower to discover Copilot Studio agents and pull their execution/usage data during the Discover phase.
