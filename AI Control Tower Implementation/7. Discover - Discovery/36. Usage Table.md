# Usage Table

> ServiceNow AI Control Tower (AICT) — Implementation Guide
> Section: Discover - Discovery → Usage Table
> Source: AI-Control-Tower.pdf, p.73

---

## What Is This?

The Usage Table is the data table that AI Control Tower's discovery process uses to record AI usage and execution data. It captures how discovered AI assets are actually being used and run. This sits alongside the Entity Tables (which store the discovered AI tools and system subcomponent relationships) as part of the discovery data model.

---

## Usage Table

| Table | Purpose |
|-------|---------|
| `sn_ai_disc_ai_usage` | Stores AI usage and execution data |

---

## How to Use This in Implementation

When implementing AI Control Tower discovery, the `sn_ai_disc_ai_usage` table is where usage and execution data captured during discovery is persisted. Implementers can query this table to inspect AI usage and execution records for discovered AI assets. It complements the entity tables (`sn_ent_ai_tool`, `sn_ent_ai_system_subcomponent_m2m`) that store the discovered tools and subcomponent relationships.
