# Connection Outcome

> ServiceNow AI Control Tower (AICT) — Implementation Guide
> Section: Discover - Discovery → Connection Outcome
> Source: AI-Control-Tower.pdf, p.67

---

## What Is This?

This subtopic describes the result of establishing an Amazon AI connection through the AI Service Graph Connector for Amazon. Once the connection is in place, AI Control Tower is able to pull AWS AI asset metadata and usage data into ServiceNow. It fits within the Discover/Discovery phase of an AICT implementation, defining what the system can do after a successful AWS connector setup.

---

## Connection Outcome

After the Amazon AI connection is created, AI Control Tower can import AWS AI asset metadata and usage data into the AI inventory and related usage tables.

---

## How to Use This in Implementation

An implementer reaches this outcome only after completing the AI Service Graph Connector for Amazon setup (required roles, AWS prerequisites, IAM permissions, and supported AWS services). Once the Amazon AI connection has been created successfully, verify that AWS AI asset metadata and usage data populate the AI inventory and related usage tables in AI Control Tower. This confirms that discovery from AWS is functioning and feeds downstream visibility into AI activity, adoption, and value realization.
