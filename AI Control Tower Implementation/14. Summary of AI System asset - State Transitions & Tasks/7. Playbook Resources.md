## What Is This?

Playbook Resources is part of the "Resources - Workflow Automation Center of Excellence" reference list in the AI System asset state transitions and tasks section. It points implementers to external ServiceNow learning and best-practice material for building and operating Playbooks used in the AI Asset Onboarding workflow. It complements the related Flows and Sub-Flows resource list on the same page.

## How to Use This in Implementation

When extending or customizing the AI Asset Onboarding Playbook, use the Playbook Resources links (Playbook best practices, Getting started, Playbooks Academy, ServiceNow Trainings) as the learning and best-practice references for designing playbook phases and tasks. Pair these with the Flows and Sub-Flows resources when the implementation also involves Flow Designer flows/sub-flows behind the playbook tasks.
