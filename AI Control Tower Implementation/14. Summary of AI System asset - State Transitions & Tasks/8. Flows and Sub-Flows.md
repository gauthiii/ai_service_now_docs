## What Is This?

"Flows and Sub-Flows" is a resource list under the Workflow Automation Center of Excellence, pointing implementers to ServiceNow Flow Designer reference material. It supports building and customizing the flows and sub-flows used to drive AI System asset state transitions and tasks during onboarding and offboarding.

## How to Use This in Implementation

When extending or customizing the AI System asset state transitions and tasks (for example, adding new tasks to a particular phase of AI Asset Onboarding/Offboarding), use these Flow Designer references to follow best practices, get started with Flows, and apply ServiceNow recommendations. These resources are intended as supporting guidance for implementers building flows and sub-flows in Workflow Studio / Flow Designer.
