## What Is This?

Native data sources are the built-in Now Platform mechanisms for populating the AI asset inventory in AI Control Tower without manual field mapping. For organizations already on the Now Platform, they offer the most seamless onboarding path: Record Producers guide asset owners through structured intake forms, while the Now Assist sync job automatically populates the AI asset inventory from existing AI skills.

---

## Native Sources and What They Populate

For organizations already on the Now Platform, native data sources offer the most seamless path with no manual field mapping required. Record Producers guide asset owners through structured intake forms, while the Now Assist sync job automatically populates the AI asset inventory from existing AI skills.

| Native Source | What It Populates |
| --- | --- |
| Request an AI System | Creates an AI System record via the AI System Record Producer. |
| Request an AI Use Case | Creates an AI Use Case intake record linked to a system. |
| Request an AI Model | Creates an AI Model record linked to a system. |
| Request a Dataset | Creates an AI Dataset record and associates it with a model. |
| Sync Now Assist AI Assets (Scheduled Job) | Automatically populates AI models, datasets, prompts, and agentic AI assets from Now Assist AI Skills. Requires the AI Control Tower for Now Assist plugin (sn_aict_nowassist). |

### Tip

MCP server assets can also be added natively via the AI Control Tower workspace. This capability was introduced in recent releases and surfaces AI gateway configurations directly within AICT.

---

## How to Use This in Implementation

When the organization is already on the Now Platform, prefer native data sources to avoid manual field mapping. Use the Record Producers (Request an AI System, Request an AI Use Case, Request an AI Model, Request a Dataset) to let asset owners populate the inventory through structured intake forms, maintaining the linkage between systems, use cases, models, and datasets. To automatically populate AI models, datasets, prompts, and agentic AI assets from Now Assist AI Skills, enable the Sync Now Assist AI Assets scheduled job, which requires the AI Control Tower for Now Assist plugin (sn_aict_nowassist). MCP server assets can be added natively through the AI Control Tower workspace.
