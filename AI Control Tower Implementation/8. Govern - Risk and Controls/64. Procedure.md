## What Is This?

This is the Procedure for working with Control Indicators in the Govern – Risk and Controls practice. Indicators are metrics used to monitor risk exposure or control performance over time, collecting data to monitor controls and risks and to gather audit evidence. This procedure lists the actions an implementer takes to create and manage control indicators.

## How to Use This in Implementation

Use this procedure when configuring control indicators to support ongoing oversight and early warning. While indicators can be created at any stage of the control lifecycle (including the Draft stage), the Monitor stage—where controls are actively implemented and ready for assessment—is the most logical time to begin using control indicators and generating indicator tasks. Follow the linked steps to first Create a Control Indicator, then Manage that Control Indicator.
