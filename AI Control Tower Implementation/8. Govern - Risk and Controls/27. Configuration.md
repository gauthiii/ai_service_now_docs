## What Is This?

This is the Configuration subsection for the Regulatory Agencies capability within the Govern - Risk and Controls area. It documents whether any additional setup or configuration is required to use Regulatory Agencies, along with related notes on how agencies are linked to authority documents. In this case, no dedicated configuration is required.

---

## Configuration

Not Applicable

### Note

- In Authority Document, use the form field "Regulated By" to build a relation with the Agency
- 360-degree view association is not supported

---

## How to Use This in Implementation

No specific configuration steps are required to use Regulatory Agencies. Instead of a standalone configuration, an implementer establishes the relationship between an Authority Document and its Agency by populating the "Regulated By" form field on the Authority Document record. Note that the 360-degree view association is not supported for this relationship.
