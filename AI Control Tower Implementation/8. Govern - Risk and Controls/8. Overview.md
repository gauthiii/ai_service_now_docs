## What Is This?

This Overview describes the AI System / Use Case Registry, which is the foundational record type within the AI Artifacts Informational Records of the AI Risk and Compliance solution. It is a central inventory of every AI-enabled business use case in the organization. It serves as the starting point for governance, risk, and compliance activities in implementation.

---

## AI System / Use Case Registry Overview

The AI System / Use Case Registry is a central inventory of all AI-enabled business use cases within the organization.

### Key Value Benefits

It provides visibility, ownership, and accountability for each AI system and serves as the starting point for governance, risk, and compliance activities.

Example: An "Automated Loan Approval System" used by the Retail Banking entity to support customer credit decisions.

## How to Use This in Implementation

Implementers begin AI governance by registering each AI-enabled use case in the AI System / Use Case Registry, establishing a single source of truth for ownership and accountability. Each registry entry (for example, an "Automated Loan Approval System") becomes the anchor for subsequent risk assessment and compliance activities. Use the "Request an AI use case / system" procedure to create new entries, then proceed to configuration and downstream governance, risk, and compliance processes.
