## What Is This?

This subtopic outlines the high-level procedure for working with risk assessments within the Govern – Risk and Controls area of ServiceNow AI Control Tower. It lists the sequence of activities an implementer follows to create, copy, score, and automate assessments. Each step links to its detailed instructions elsewhere in the guide.

---

## Procedure Steps

The procedure consists of the following ordered activities, each referenced by a link in the source document:

- **A.** Create Assessment Template (link)
- **B.** Copy Assessment Template (link)
- **C.** Scoring Assessment (link)
- **D.** Automate Response (link)
- **E.** Post Assessment Automation (link)

---

## How to Use This in Implementation

An implementer applies this procedure as the end-to-end workflow for risk assessments: begin by creating an assessment template, optionally copy an existing template to reuse it, configure scoring for the assessment, automate the response handling, and finally set up post-assessment automation. Follow the linked detail sections for each step in sequence, since later steps (scoring, automation) depend on the template established in the earlier steps.
