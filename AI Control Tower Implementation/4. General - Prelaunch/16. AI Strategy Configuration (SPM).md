## What Is This?

AI Strategy Configuration (SPM) covers the prelaunch steps for using Strategic Portfolio Management (SPM) to manage AI strategy, intake, and execution within AI Control Tower. It connects AICT to SPM's Goals Framework and Strategic Planning capabilities so that AI-related strategies, goals, targets, demands, and work can be planned and tracked. The Goals Framework is included out-of-the-box for all AICT customers, while the Strategic Planning plugin requires an SPM Pro license.

---

## AI Strategy (SPM) Configuration

If using Strategic Portfolio Management for AI strategy, intake, and execution, complete the following steps. The Goals Framework is included out-of-the-box for all AICT customers. The Strategic Planning plugin requires an SPM Pro license.

| ✓ | AI Strategy (SPM) |
|---|---|
| ☐ | Verify the Goals Framework is active (included out-of-the-box for all AICT customers) |
| ☐ | Test creating a Strategic Priority, Goal, and Target from the AICT List section |
| ☐ | If SPM Pro is licensed: activate the Strategic Planning plugin |
| ☐ | Test creating a Demand with Investment Type set to Artificial Intelligence |
| ☐ | Verify that Demands can be converted into Projects and Epics |
| ☐ | Verify that the Strategic Planning Workspace displays AI-related work with capacity planning, financial analysis, and scenario planning |
| ☐ | Verify that the AI Strategy tab in the AICT Workspace displays strategies, goals, targets, costs, and prioritized work |

---

## How to Use This in Implementation

Implementers using SPM for AI strategy should first confirm the Goals Framework is active, since it ships out-of-the-box with AICT and enables creating Strategic Priorities, Goals, and Targets directly from the AICT List section. If the organization holds an SPM Pro license, activate the Strategic Planning plugin to unlock demand intake (using the Artificial Intelligence Investment Type), conversion of Demands into Projects and Epics, and the Strategic Planning Workspace (capacity planning, financial analysis, scenario planning). Finally, validate that the AI Strategy tab in the AICT Workspace surfaces strategies, goals, targets, costs, and prioritized work. The key dependency/gotcha: Strategic Planning features beyond the Goals Framework require the SPM Pro license.
