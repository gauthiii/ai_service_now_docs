## What Is This?

The Value section of AI Control Tower explains how AICT quantifies the business value delivered by AI assets. It introduces the core value metrics that AICT calculates to measure AI usage and benefit. This sits within the Govern - Risk and Control area and feeds the Value dashboards used to demonstrate the impact of AI investments.

## How to Use This in Implementation

To apply the Value capability, implementers rely on AICT's value calculation engine, which derives metrics from AI asset usage data. Begin by ensuring AI usage is being recorded, then review how the core value metrics are computed before configuring Value dashboards. Because page 252 only introduces the metric framework ("AICT calculates value using three primary metrics"), refer to the subsequent pages for the full metric definitions and configuration details (such as time-saved values and acceptance-rate sources).
