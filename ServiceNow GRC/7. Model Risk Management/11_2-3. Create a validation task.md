Create a validation task
Create a validation task by selecting a predefined validation template, assigning a validator, and
setting a due date. This task defines the validation scope and responsibilities, preparing the
model for formal validation and deployment.
Before you begin
Role required: sn_model_risk_mgmt.model_governance or
sn_model_risk_mgmt.risk_coordinator
About this task
You can create a validation task for a model when it is in the Pre-deployment or Monitor stage.
Procedure
1.Navigate to All > Model Risk Workspace.
2. Select the list icon .
3. Select the model for which you want to create a validation task.
4.In the Pre-deployment state, select Create task.
5. On the Model Validation dialog, fill in the fields.
