Link existing documents to a model record
Link documents from your repository to a model record. This feature helps you manage all model-
related documents in one place, ensuring version control, audit readiness, and traceability.
Before you begin
Role required: sn_model_risk_mgmt.executive_reader, sn_model_risk_mgmt.model_validator,
sn_model_risk_mgmt.model_risk_coordinator, sn_model_risk_mgmt.model_governance
Users on the Xanadu and Yokohama releases must install the Document Management plugin
manually. Users upgrading from an earlier release to Zurich must also install it manually. For new
installations, the plugin is installed automatically.
Procedure
1.Navigate to All > Model Risk Workspace.
2. Select the list icon .
3. From the list, navigate to Models > All.
4.Select the model and navigate to the Documents tab.
5. Select Link Documents.
