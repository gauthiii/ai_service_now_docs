Create an issue for model
Add or create model issues and their remediation actions. Capture validation findings or
performance problems and follow them through to resolution.
Before you begin
Role required: sn_model_risk_mgmt.model_risk_coordinator,
sn_model_risk_mgmt.model_governance
Procedure
1.Navigate to All > Model Risk Workspace.
2. Select the list icon .
3. From the list, navigate to Models > All.
4.Select the model and navigate to the Issues tab.
5. Select New.
6.On the form, fill in the fields.
For a description of the field values on the Create an issue form, see Create an issue form.
7.Select Save.
Create an issue form
Use the Create an issue form to capture validation findings or performance problems for a model.
See the following table for a description of the field values.
