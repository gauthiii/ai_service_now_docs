Compliance case task workflow
You can collaborate with multiple teams to investigate, perform an impact assessment, and
gather the evidence to capture the details and responses for review in the GRC: Compliance
Case Management application.
After a compliance case is reported, the compliance analyst can create case tasks and assign
those tasks to the various case task owners. The case task owners can then work on the tasks
and submit them for the case analyst to review. There are three types of case tasks:
Assessment
An Assessment case task requires the case task owner to take an assessment and
provide responses. A case task of this type can only be reassigned.
Investigation
An Investigation case task requires the case task owner to provide observations.
Others
An Others case task also requires the case task owner to provide observations.
After the case task owners work on a task, they submit the task for the case analyst to review.
The following example displays the life cycle of a case task.

About this task
After a case is reported to the compliance team, a compliance case analyst can create case
tasks and assign the tasks to the relevant task owners. This way, the compliance case analyst
can collaborate with various teams to investigate, assess, and capture observations that are
related to the case.
Procedure
1.Navigate to All > Policy and Compliance > Compliance Workspace.
2. Select the list icon.
3. In the List section, select All Cases.
4.Open the case that you want to create an action task for.
5. Select the Action task related list.
6.Select New.
7.On the form, fill in the fields.
For a description of the field values on the action task form, see Case task form.
8.Select Save.
9.To assign the task to the Action task owner, select Assign task.
10.Optional: In the confirmation dialog box, provide your notes.
11. In the confirmation dialog box, select Move to assigned.
The state of the record changes to Assigned and the action task is assigned to the Action task
owner.
12. Optional: To reassign the action task to another user before the action task is accepted by the
action task owner that is specified by you, select Reassign.
Note: You can only reassign the Assessment type of case tasks.
a.In the Assignment group field, select the relevant group.
b.In the Action task owner field, select a new owner.
c.In the Reassign Action task dialog box, select Reassign.

Work on an action task
Accept the Action task, provide the details requested by the case analyst, and submit the Action
task for a review in the GRC: Compliance Case Management application. When an action task is
assigned to an assignment group, any user from the assignment group can accept and work on
the task.
Before you begin
Role required: sn_compliance_ws.corporate_compliance_analyst
About this task
If an action task is the Assessment type, the Action task owner must take an assessment to
determine the losses that were incurred from a case. In this case, the task owner can accept the
work and then take the assessment. If the Action task is the Investigation type or Others type, the
Action task owner can accept the task, provide observations, and submit the case for review to
the case analyst.
Procedure
1.Navigate to Self Service > Employee Center.
2. Access your task from the My Tasks module.
3. Open the task that is assigned to you from Action Tasks.
4.In the Open section, select Accept work.
5. In the confirmation dialog box, select Accept work.

Reassign an Assessment type Action task
Reassign an Assessment type Action task to another user in the GRC: Compliance Case
Management application. For example, a case analyst or an action task owner can reassign a
compliance Action task to another Action task owner if an action task is wrongly assigned or isn’t
accepted by the Action task owner.
Before you begin
Role required: sn_compliance_ws.corporate_compliance_analyst
About this task
As a case analyst, after a task is assigned to an action task owner, you can reassign the task to
another user.
As an action task owner, you can also accept the task, start working on it, and then reassign the
partially completed task to another user. When you reassign a partially completed Action task
to another user, the data that you provide also gets copied and is available to the new Action

task owner. The newly assigned task owner can modify the Action task based on that owner's
requirements.
Note: You can only reassign the case tasks that have the Type field set to Assessment.
Procedure
1.Navigate to Self Service > Employee Center.
2. Access your task from the My tasks module.
3. Open the task that you want to work on.
4.In the Open section, select Reassign.
The Reassign Action task dialog box appears.
5. In the Assignment group field, select the assignment group.
6.In the Action task owner field, select the owner.
7.In the Reassign Action task dialog box, select Reassign.
