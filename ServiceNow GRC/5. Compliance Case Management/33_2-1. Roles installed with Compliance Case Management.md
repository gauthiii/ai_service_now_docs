Roles installed with Compliance Case Management
The GRC: Compliance Case Management installs the essential role to perform respective day-
to-day operational tasks towards managing compliance cases for the enterprise to perform their
respective tasks.


| Table | Description |
| --- | --- |
| [sn_grc_case_mgmt_cause] |  |
| Cause and consequence [sn_grc_case_mgmt_cause_consequence] | Stores the causes and consequences that were mapped to the case. |
| Consequence [sn_grc_case_mgmt_consequence] | Stores the consequences that were added to the library and mapped to the case. |
| GRC core case [sn_grc_case_mgmt_core_case] | Stores the core functionality details of the case. |
| Impacted area [sn_grc_case_mgmt_impacted_area] | Stores the impacted areas that were mapped to the case. |

| Regulation [sn_grc_case_mgmt_regulation] | Stores the regulations that were mapped to the case. |
| Regulated area [sn_grc_case_mgmt_related_area] | Stores the related areas that were mapped to the case. |
| Compliance case [sn_comp_case_compliance_case] | Stores the compliance case records. |
