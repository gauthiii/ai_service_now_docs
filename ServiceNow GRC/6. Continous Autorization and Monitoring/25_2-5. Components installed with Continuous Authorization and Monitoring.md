Components installed with Continuous Authorization and Monitoring
Activating the GRC: CAM plugin adds or modifies several tables, user roles, and other
components.
Continuous Authorization and Monitoring system properties
Default properties are set with activation of CAM plug-in.
To configure the CAM system properties, navigate to All > Continuous Authorization
and Monitoring > Administration > System Properties.
