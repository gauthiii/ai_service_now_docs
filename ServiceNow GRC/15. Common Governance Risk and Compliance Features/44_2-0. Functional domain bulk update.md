Functional domain bulk update
The functional domain feature enables users to efficiently filter and sort records within
a workspace based on domain-specific tags. For instance, when working in the Privacy
Workspace, users can exclude records that are not tagged with privacy, allowing for a more
focused view and reducing potential confusion.
The bulk update functionality enables users to apply or remove functional domains to multiple
records simultaneously.
Manual and automatic tagging of records
Tagging methods: Users can tag records with functional domains using two methods: manual
and automatic.
