Workspace page configuration
A common record page can be used and configured within all the GRC applications.
A common record page template is introduced to customize tables in any GRC workspaces.
You can configure the record page to show icons, separator, and stepper component. For more
information on how to configure the workspace, see Configure workspace page topic.
Configure workspace page
You can configure and customize your workspaces so that all the GRC applications have a similar
appearance.
Before you begin
Role required: sn_grc_workspace.admin.
Procedure
1.Navigate to All > GRC Administration > Workspace Page Configuration.
2. Select the record that you want to configure.
3. Open the record and on the form, select here to edit the record.
On the form, fill in the fields. For more information, see Workspace page configuration form.

Revert record page templates to the pre-17.x version
If you’re an existing customer and have a customized record page, you can revert to that old
page.
Before you begin
Role required: workspace_admin
Procedure
1.Navigate to All > Now Experience Framework > UI Builder.
2. Select the workspace in which you want to make changes, from the My Experiences list.
3. Select Record page that belongs to your workspace from the list.
4.In the record page, select Developer listed under Menu tab and select Open variant
collection.
UX screen variants
In the UX Screens tab, you see two screens, Record default and Risk record page.
