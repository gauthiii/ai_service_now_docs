Before you begin
Role required: sn_grc_workspace.admin
About this task
Notification routing rules determine which workspace a user is directed to when they select
a notification link. You create routing rules by adding records to the notification redirection
metadata table. Each rule maps a table to a target workspace with an order value that
determines evaluation priority. If a user has access to only one workspace for a table, the system
automatically directs them to that workspace. Routing rules are used only when a user has
access to multiple workspaces for the same table. In this case, the system directs them to the
workspace with the lowest order value they can access.
