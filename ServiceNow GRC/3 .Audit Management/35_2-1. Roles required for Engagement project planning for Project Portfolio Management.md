Roles required for Engagement project planning for Project Portfolio
Management
The advanced planning capability enables integration of Advanced Audit with the Project
Portfolio Management (PPM) product. The capability creates an equivalent project for the
engagement, and provides planning features that could be used for resource planning, cost
planning, and so on, from the engagement project.
