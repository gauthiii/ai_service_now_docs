Setup for the UI Builder
You can set up and extend the BCM Workspace pages and components by using the UI Builder.
About UI Builder
UI Builder is a drag-and-drop Web user interface builder that uses the Next Experience
Components and custom web Components.
To build and customize your Workspace pages and components, you must have the
ui_builder_admin role.
