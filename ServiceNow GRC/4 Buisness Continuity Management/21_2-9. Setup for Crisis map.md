With the BCM administrator role, you can perform the administrative tasks to set up the Crisis
map:
•Scheduled Imports: Configure a Scheduled Data Imports record for the Crisis map application.
For more information, see Configure Scheduled Data Imports records.
•Resource Configuration: Configure a Resource Configuration record for the Crisis map
application. For more information, see Configure Resource Configuration records.
•Alert Rules: Configure an alert rule in Crisis map so that you can specify the conditions
under which a feed can be displayed in the dashboard as an alert. For more information, see
Configure alert rules.
•Alert Actions: Configure an alert action in UIB Workspace for the Crisis map application. For
more information, see Configure alert actions.

You can dismiss conditions when a feed is deleted or resources are not impacted. The
example shows how to dismiss a condition in an alert rule.
